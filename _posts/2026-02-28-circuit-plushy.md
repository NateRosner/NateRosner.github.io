Description of the Project:

  This project requires the use of Lilypad components, conductive thread, felt, and normal embroidery thread in order to create a "plushy" that lights up when the Lilypad switch is on and the Lilypad button is pressed. By connecting the correct ends of each component (+ and -) with a running stitch of conductive thread, we are able to create a channel of components that is connected to a battery that turns on the three LED lights. Using sewing techniques, we then create a shape that is sewn together with a whip stitch in order to create a shaped object that uses circuits to light up LED lights on the plushy.

One Tip/Trick:

  One tip I have is to really map out what you are doing before you actually do it (ie. put time into the paper prototype and alligator prototype). This way, when it comes time to sew the components onto the plushy, it is alot easier to focus on the sewing itself rather than worrying about if the components will work.
