The Plushy Redux programming assignment uses the int btn and swt to show the components of the plushy that turned all three LEDs on. Using if statements, we can determine that the desired behavior of the three led lights is that they only turn on when the swtich is on and the button is pressed. If either of these are untrue, no led lights will turn on. By making an if statement that only when btn and stw are on, then all three led lights turn off. If anything else is plugged into the integer values, no led lights will turn on.
