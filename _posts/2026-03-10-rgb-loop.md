This program uses a while loop as well as nested if.. then statements to print a color on the 10th cycle of the loop. However, the challenge is to print a color three times in a row in the cylcle before moving to the next cycle. Example: R,R,R,G,G,G,B,B,B,R,R... 

VS Code Debug Console:

![rgbloop](https://naterosner.github.io/assets/img/rgbloop.png)

One tip/trick I have is to actually use logic before typing anything. I spent time with just two variables (int runtimes and int index). I realized soon that i needed another variable that allowed me to correctly program the number of time and when to print the correct color in the sequence. I decided to use numprints (ie. Number of Prints). I used a nested if then statement to show that when every time on the 10th time through the loop, it would print a color. But nesting numprints actually made it easy to show that once it printed a number three times, it would move to the next and so on. Also, making sure to add numbprints = numprints + 1 and if (numprints == 9) made sure it actually looped back to the orignal numprints and color (0 and RED).
